# Signal Scorecard

Public accountability ledger for every signal posted on the feed. Every
D+5 outcome vs SPY, wins AND losses, updated daily on weekdays at 14:50 PT.

**Live page**: https://dhawalc.github.io/signal-scorecard/

Methodology: every posted ticker's close-to-close return is computed at
D+1/5/10/20 trading days and benchmarked against SPY over the same window.
Alpha = ticker return − SPY return. Source data: yfinance. Not financial advice.

Source: this page is regenerated and pushed by a cron on the operator's
local machine; no manual edits.
